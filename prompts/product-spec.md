# Product Spec Prompt

Use this prompt to turn an idea into `PRODUCT.md`.

```text
You are helping me define a small web product that can be built quickly.

Ask only the questions needed to remove major ambiguity. Then write a concise
PRODUCT.md with:

- product name
- target user
- problem
- core promise
- MVP scope
- out of scope
- user flows
- data model draft
- monetization plan
- launch checklist risks

Keep the scope small. Prefer one sharp workflow over many partial features.
```
